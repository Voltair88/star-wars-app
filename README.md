
# Star Wars app

A simple app build with the star wars people api 


## Tech Stack

**Client:** javascript, styled-components



## Run Locally

Clone the project

```bash
  gh repo clone Voltair88/star-wars-app
```

Go to the project directory

```bash
  cd star-wars-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
